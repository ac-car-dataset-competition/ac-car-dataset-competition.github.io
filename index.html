AC - Car price prediction competition

Goal

This competition's goal is to predict with the best accuracy, the prices of an unknown car dataset. In order to do so, you'll be provided a training dataset, which you can use to tune your parameters.

Rules

The algoritm can be done in any language of your preference, but it must be well written so it can be understood by people with few knowledgement of the given language

It must take a csv file (using comma, not semicolon) as input, containing the following fields (the file contain headers):

  • name
  • seller
  • offerType
  • abtest
  • vehicleType
  • yearOfRegistration
  • gearbox
  • powerPS
  • model
  • kilometer
  • fuelType
  • brand
  • notRepairedDamage
  • dateCreated
  • nrOfPictures
  • postalCode
  • lastSeen

And return a csv file with the predicted price for each line

PS: The input and the output file must have the same amount of lines and each line in the output file must match the line in the input file
